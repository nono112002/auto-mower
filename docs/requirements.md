# 自動草刈りロボット 要件書

最終更新: 2026-04-05

---

## なぜやるか（動機・背景）

農地の畝間通路の雑草管理は労力がかかる繰り返し作業である。
自動化することで省力化を図り、同時に自律走行ロボットの設計・製作をポートフォリオとして記録する。

---

## 何を知りたいか（問い）

- 前後アッカーマンステアリングは農業ロボットの自律走行に実用的か
- GPSによる畝間直線追従はどの程度の精度で実現できるか
- チップソーを車体内に収める設計で畝際まで草刈りできるか

## 使用環境・前提条件

| 項目 | 内容 |
|---|---|
| 対象 | 畝間通路の雑草（草刈り、芝刈りではない） |
| 地形 | 平坦地（斜面対応は将来課題） |
| 運用 | GPS自律走行 |
| 畝幅 | 600mm |
| 畝間通路幅 | 500mm（ロボット外形480mm + GPS誤差マージン） |

## 車体仕様

### 外形・寸法

| 項目 | 寸法 | 備考 |
|---|---|---|
| 車体幅 | **480mm** | チップソー外縁 = タイヤ外縁 |
| 車体長 | TBD（目安 600〜800mm） | ホイールベース確定後 |
| ホイールベース | TBD（目安 350〜400mm） | チップソー前後クリアランス確保 |

### ステアリング方式

**前後アッカーマン（4輪ステア）**

- 前後逆相（カウンターステア）で最小旋回半径を確保
- 前後同相でクラブ走行も将来対応可能

| 項目 | 値 | 算出根拠 |
|---|---|---|
| 路線間距離 | 1100mm | 畝間通路幅500 + 畝幅600 |
| 必要実効旋回半径 | **≤ 550mm** | 路線間距離 / 2 |
| 機構設計目標（前後逆相） | ≤ 1100mm | 実効半径の2倍 |

### 駆動方式

- スキッドステア（左右独立駆動）または4輪独立駆動

## 草刈り仕様

### ブレード配置

```
[左前輪]          [右前輪]
    |←──── 480mm ────→|
    |  ⊙チップソーL    |
    |       20mm gap   |
    |  ⊙チップソーR    |
    |                  |
[左後輪]          [右後輪]
```

- チップソー外縁とタイヤ外縁を一致させ、畝際まで草刈り可能にする
- タイヤはホイールベースの前後に配置し、チップソーとの干渉を回避

### ブレード仕様

| 項目 | 仕様 |
|---|---|
| ブレード種類 | チップソー |
| ブレード径 | 230mm × 2枚 |
| ブレード間隔 | 20mm |
| 刈り幅 | 480mm（全幅） |
| 取付方式 | ホイールベース間に搭載 |

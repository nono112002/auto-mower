## 設計方針

- Ver1はシンプルに作って走らせることを優先する
- **アーティキュレートステアリング（前後フレーム分割・センターピン）を本命とする**（スクラブ少・草地ダメージ小）
- 自律走行はIMU＋オドメトリ、サブでラジコン操作（DUALSHOCKコントロール）
- マイコン（Pico W）を走行モジュールとして独立設計し、Ver2以降でラズパイと接続できる構造にする
- モーター出力シャフトに車体荷重を負わせない（ベアリングで別支持）
- チップソーは前輪軸中央配置（前輪タイヤ内側に横方向49mmクリアランス確認済み）

### Ver2以降の検討事項（Ver1では実装しない）
- ラズパイ追加（GPS / AI等の高度処理）
- RTK GPS（ZED-F9P + モバイルルーター + NTRIP）
- フローティングデッキ（チップソーの地面追従）
- ホイール素材アップグレード：PETG → PETG-CF or ASA
  - PETG-CF：[Bambu Lab ストア](https://jp.store.bambulab.com/products/petg-cf)
  - ノズル（A1用ハードニングスチール）：[Bambu Lab ストア](https://jp.store.bambulab.com/collections/a1-accessories)

### Ver1

```
[DUALSHOCK4]  ──→┐
[MPU9250]     ──→├── [Pico W] ──→ [TB6612FNG] ──→ [駆動モーター×4]
[エンコーダー] ──→┘         └──→ [ステアリングアクチュエーター]
```

### Ver2以降

```
[ラズパイ（GPS / AI等）]
        ↓ RS485（有線）
[Pico W（走行モジュール）] ──→ [TB6612FNG] ──→ [モーター×4]
[MPU9250]      ──→ ↑
[エンコーダー] ──→ ↑
```

Pico Wは「足回りの専任モジュール」として独立。ラズパイはVer2以降で上位コントローラーとして追加。

### トルク・速度計算根拠

```
転がり抵抗係数（草地）: μ_rr = 0.07
ロボット推定重量:       20kg
全体必要駆動力:         20 × 9.8 × 0.07 = 13.7N
1輪あたり（4WD）:       13.7 / 4 = 3.4N
ホイールトルク:         3.4N × 0.170m = 0.58Nm ≈ 6 kg·cm（連続）
安全率3倍 → ストールトルク目標: 18 kg·cm → 24 kg·cm ✅

目標速度: 2 km/h = 0.556 m/s
ホイール回転数: 0.556 / (2π × 0.170) ≈ 52 RPM
負荷時推定: 58 × (1 - 6/24) ≈ 43 RPM → 0.77 m/s = 2.8 km/h ✅
```

### 動力伝達方式：Vベルト（確定）

ベアリング内蔵一輪車ホイールを使用し、固定シャフトをナックル代わりとする。モーターはフレーム上部のアームに配置し、Vベルトでホイールプーリーを駆動する。

```
フレームアーム上部
  └── JGB37-520（モーター）
           │
        Vベルト（縦方向）
           │
  ホイールプーリー（一輪車ホイール面に取り付け）
           │
  ══════════════  ← ベアリング内蔵ハブ（一輪車ホイール）
           │
  固定シャフト（フレームに固定・車体荷重を支持）
```

**モーター上方配置の理由：**
- チップソーの腹下スペース確保
- 防水・防塵（泥はねから遠ざける）
- チェーン/ベルト距離の確保（短すぎるとテンション管理が困難）

**Vベルト採用理由：**
- 給油不要・屋外環境に強い
- 農業機械での実績が豊富
- 安価・ホームセンターで入手可能
- 過負荷時のスリップがモーター保護になる

**⚠️ 懸念事項：スキッドステア旋回時のスリップ**
```
スキッドステア旋回 → 左右輪で大きなトルク差が発生
→ Vベルトがスリップする可能性あり
→ 走行テストで確認する

スリップが問題になった場合の対策：
① テンションプーリーで張力強化
② Vベルト → チェーンに変更（確実伝達・ただし給油必要）
```

```
Makita 18V バッテリー
  ├─ チップソーモーター（ESC経由・18V直結）
  │    最大電流: ~20A
  ├─ DC-DC降圧（18V → 12V）─ 駆動モーター × 4
  │    最大電流: 2.3A × 4 = 9.2A（全輪ストール時）
  │    配線: AWG16（1.3mm²）以上
  └─ DC-DC降圧（18V → 5V） ─ Pico W・センサ系
```

## 4WD レイアウト設計

詳細は [`docs/layout.md`](layout.md) を参照。

**主要確定値：**
- ホイールベース：400mm（下限380mm）、全長：~715mm
- チップソー：前輪軸中央配置（横方向クリアランス49mm確認済み）
- バッテリー：前輪軸から+284mmに固定（前後50/50達成）
- ピボットターン：μ≤0.35で可能。乾燥草地は差動旋回（R=1〜2m）で対応

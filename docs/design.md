## 設計方針

- v1はシンプルに作って走らせることを優先する
- 4輪駆動スキッドステア（左右差動）で旋回
- 自律走行はIMU＋オドメトリ、サブでラジコン操作
- マイコン（Pico W）を走行モジュールとして独立設計し、Ver2以降でラズパイと接続できる構造にする
- モーター出力シャフトに車体荷重を負わせない（ナックル内蔵ベアリングで支持）
- チップソーはトレッド幅・ホイールベース確定後に配置を決める

### Ver2以降の検討事項（Ver1では実装しない）
- ラズパイ追加（GPS / AI等の高度処理）
- RTK GPS（ZED-F9P + モバイルルーター + NTRIP）
- センターピボット（前後フレーム分割による4輪接地保証）
- フローティングデッキ（チップソーの地面追従）
- 旋回補助ステアリング（スキッドターンの土削り軽減）
- ホイール素材アップグレード：PETG → PETG-CF or ASA
  - PETG-CF：[Bambu Lab ストア](https://jp.store.bambulab.com/products/petg-cf)
  - ノズル（A1用ハードニングスチール）：[Bambu Lab ストア](https://jp.store.bambulab.com/collections/a1-accessories)

### トルク・速度計算根拠

```
転がり抵抗係数（草地）: μ_rr = 0.07
ロボット推定重量:       20kg
全体必要駆動力:         20 × 9.8 × 0.07 = 13.7N
1輪あたり（4WD）:       13.7 / 4 = 3.4N
ホイールトルク:         3.4N × 0.170m = 0.58Nm ≈ 6 kg·cm（連続）
安全率3倍 → ストールトルク目標: 18 kg·cm → 24 kg·cm ✅

目標速度: 2 km/h = 0.556 m/s
ホイール回転数: 0.556 / (2π × 0.170) ≈ 52 RPM
負荷時推定: 58 × (1 - 6/24) ≈ 43 RPM → 0.77 m/s = 2.8 km/h ✅
```

### 動力伝達方式：Vベルト（確定）

ベアリング内蔵一輪車ホイールを使用し、固定シャフトをナックル代わりとする。モーターはフレーム上部のアームに配置し、Vベルトでホイールプーリーを駆動する。

```
フレームアーム上部
  └── JGB37-520（モーター）
           │
        Vベルト（縦方向）
           │
  ホイールプーリー（一輪車ホイール面に取り付け）
           │
  ══════════════  ← ベアリング内蔵ハブ（一輪車ホイール）
           │
  固定シャフト（フレームに固定・車体荷重を支持）
```

**モーター上方配置の理由：**
- チップソーの腹下スペース確保
- 防水・防塵（泥はねから遠ざける）
- チェーン/ベルト距離の確保（短すぎるとテンション管理が困難）

**Vベルト採用理由：**
- 給油不要・屋外環境に強い
- 農業機械での実績が豊富
- 安価・ホームセンターで入手可能
- 過負荷時のスリップがモーター保護になる

**⚠️ 懸念事項：スキッドステア旋回時のスリップ**
```
スキッドステア旋回 → 左右輪で大きなトルク差が発生
→ Vベルトがスリップする可能性あり
→ 走行テストで確認する

スリップが問題になった場合の対策：
① テンションプーリーで張力強化
② Vベルト → チェーンに変更（確実伝達・ただし給油必要）
```

### 干渉チェック根拠

タイヤ内縁：480/2 − 76/2 = 202mm（タイヤ中心）− 38mm（タイヤ半幅）= **164mm**  
チップソー外縁：**115mm**  
**横方向クリアランス：164 − 115 = 49mm ✓**

前後方向：チップソーと前輪は横方向に49mm離れているため、同一前後位置でも干渉なし ✓

### ⚠️ 懸念事項：前輪軸上のモーター集中

前輪軸付近にチップソーBLDCモーター（ø65mm）と前輪駆動JGB37-520が集中する。

**対策候補：**
1. **高さ分離**：CSモーターを低く（地面から100mm）、駆動モーターをアーム上部（200mm+）に配置
2. **CSを50mm前出し**：前輪軸より50mm前方にCS中心をずらす（全長+50mm、チップソー前端165mm前方）

→ Ver1設計では対策①を基本方針とし、3Dモデル確認後に調整。

### スキッドステアターン性能計算

**ピボットターン（その場旋回）の必要トルク：**

```
T_turn = μ × W × d_wheel
d_wheel = √((WB/2)² + (Tread/2)²) = √(200² + 202²) ≈ 284mm

モーター最大ターントルク:
T_motor_max = 2 × (2×ストール24kg·cm / R_tire170mm) × (Tread/2)
            = 2 × 28.2N × 0.202m = 11.4 N·m
```

**車重とターン可能条件：**

| μ（草地） | ピボットOK最大車重 | 備考 |
|---|---|---|
| 0.2（湿った軟草） | 20.1 kg | |
| 0.3（標準草地） | **13.4 kg** | 実重量12kgなら✅ |
| 0.4（乾燥硬草） | 10.0 kg | 12kgでは❌ |

**実重量再試算（詳細）：**

| 部品 | 概算重量 |
|---|---|
| アルミフレーム | ~1,500g |
| JGB37-520 × 4 | ~1,200g |
| Vベルト・プーリー・ベアリング | ~600g |
| ノーパンクタイヤ × 4 | ~3,200g |
| PETGホイール × 4 | ~600g |
| BLDC + ESC | ~350g |
| チップソー刃 + マウント | ~400g |
| バッテリー Makita 6Ah × 2 | ~1,800g |
| 基板・配線 | ~400g |
| ボルト類 | ~400g |
| **合計** | **≈ 10.5 kg** |

設計仕様「~20kg」は過大。**実態は10〜12kg程度**。

**WBを短くする・タイヤを小さくすることの効果：**

| WB | d_wheel | T_turn改善 |
|---|---|---|
| 400mm（現状） | 284mm | 基準 |
| 300mm | 252mm | -11% |
| 200mm | 225mm | -21% |

→ WBを大幅に短くしても改善は最大21%止まり。根本解決にならない。  
→ タイヤ径を小さくすると乗り上げ限界が下がる（3.00-8: ~85mm → 3.00-4: ~63mm）ためコスト大。非推奨。

**⚠️ 設計の割り切り：**  
乾燥した固い草地（μ≥0.4）でのピボットターンは困難。  
→ **行端転換は差動旋回（R=1〜2m）または3点ターンで対応。ピボットターンは不要件とする。**

### 重心（CG）・バッテリー配置

**バッテリーなし本体 CG：前輪軸から +181mm（19mm前寄り）**

50/50バランス達成条件（WB=400mm → 中点200mm）：

| バッテリー | 質量 | 配置位置（前輪軸から） | 走行時間概算 |
|---|---|---|---|
| Makita 互換 6Ah × 2 | ~1,800g | +284mm（後輪116mm手前） | ~2時間 |
| Makita 互換 9Ah × 2 | ~2,600g | +250mm（後輪150mm手前） | ~3時間 |

→ どちらも前後軸間に収まる配置でバランス可能。  
→ **フレームのバッテリーマウントは±50mm調整できるスロット構造にする**（CG微調整のため）。

### フォールバック判断基準

以下のいずれかが解決不能な場合、差動二輪＋リアキャスターに移行する：
- 前輪軸上のパッケージング（モーター競合）が物理的に解決不能
- 全長が850mmを超えて畝間通路への進入が困難になる
- Vベルトのスリップが制御不能でチェーン化しても問題が残る

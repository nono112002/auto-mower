## チップソー配置：前輪軸中央配置（採用）

チップソーを前輪タイヤの横方向中央・前輪軸と同位置に配置することで、  
全長を最小化しつつ「刃が車両最前端」の要件を満たす。

```
【上面図】
         ←────────── 480mm ──────────→
         ┌──────────────────────────────┐
  前 →   │  [M左前]  [CS]  [M右前]      │  ← モーター（上部アーム）
         │  ┌───┐  ┌──○──┐  ┌───┐  │
         │  │▓▓▓│  │ ø230 │  │▓▓▓│  │  ← タイヤ・チップソー刃
         │  └───┘  └──────┘  └───┘  │
         │  ↑前輪軸（チップソー中心も同位置）  │
         │                              │
         │      [バッテリー×2]           │
         │      [Pico W / 基板]         │
         │                              │
         │  ┌───┐            ┌───┐  │
         │  │▓▓▓│            │▓▓▓│  │  ← 後輪
         │  └───┘            └───┘  │
  後 →   │  [M左後]          [M右後]   │
         └──────────────────────────────┘

【側面図】
         [CS motor（BLDC縦置き）]
         [M]─Vベルト─[前輪]      [M]─Vベルト─[後輪]
  地面 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
         ↑CS刃（h≈40mm）←前輪軸←──400mm──→後輪軸
         CS前端が車両最前端（前輪軸より115mm前方）
```

### 干渉チェック

| 方向 | 計算 | 結果 |
|---|---|---|
| 横方向クリアランス | タイヤ内縁164mm − チップソー外縁115mm | **49mm ✅** |
| 前後方向干渉 | 横方向に49mm離れているため、同一前後位置でも干渉なし | **✅** |

```
タイヤ内縁 = 480/2 − 76/2 = 202（タイヤ中心）− 38（タイヤ半幅）= 164mm
チップソー外縁 = 115mm（半径）
クリアランス = 164 − 115 = 49mm
```

### ⚠️ 懸念：前輪軸上のモーター集中

チップソーBLDCモーター（ø65mm）と前輪駆動JGB37-520が前輪軸付近に集中する。

**対策：高さ分離**  
CSモーターを低く（地面から100mm程度）、駆動モーターをアーム上部（200mm以上）に配置。  
→ 3Dモデルで確認後に確定。

## スキッドステア旋回性能

### 分力による理解（直感的モデル）

ピボットターン（その場旋回）時、右前タイヤを例に取る。

```
               ●（車両中心）
              ╱
      d=284mm╱
            ╱ φ≈45°
           ╱
──────────╱──────────── タイヤ転がり方向（後退）
         ╲
          ╲ 接線方向（ここに力を使いたい）
```

タイヤは縦方向にしか転がれないため、モーター力 F_m のうち「回転円の接線方向」に向く分力だけが旋回に有効。残りはスクラブ（横ずれ）となる。

```
φ = arctan(WB/2 ÷ Tread/2) = arctan(WB ÷ Tread)

有効分力（接線成分）= F_m × cos(φ) = F_m × (Tread/2) / d_wheel
スクラブ成分（径方向）= F_m × sin(φ) = F_m × (WB/2) / d_wheel

d_wheel = √((WB/2)² + (Tread/2)²)
```

一方、**地面摩擦は接線方向そのものに働くため分力なしで全量が抵抗になる。**  
これがモーターと摩擦の非対称性の本質。

### 公式

```
【旋回トルク　バランス条件】

モーター寄与トルク（4輪合計）:
  τ_motor = 4 × F_motor × cos(φ) × d_wheel
          = 4 × F_motor × (Tread/2)        ← d_wheel が消える

摩擦抵抗トルク（4輪合計）:
  τ_friction = 4 × μ × (W/4) × d_wheel
             = μ × W × d_wheel             ← d_wheel はそのまま残る

ピボットターン成立条件：
  τ_motor ≥ τ_friction
  4 × F_motor × (Tread/2)  ≥  μ × W × d_wheel

最大旋回可能車重：
  W_max = 2 × τ_stall × Tread / (r_tire × μ × d_wheel)
```

**モーター有効率（cos φ）：WBが大きいほど低下**

```
φ = arctan(WB ÷ Tread)
cos(φ) = (Tread/2) / d_wheel  ← モーターの実効効率
```

| WB | φ | cos(φ)（モーター効率） | スクラブ比（対差動二輪） |
|---|---|---|---|
| 0mm（差動二輪） | 0° | **1.00（100%）** | 基準（最小） |
| 200mm | 26° | 0.90 | +11% |
| 300mm | 36° | 0.81 | +25% |
| **400mm（現状）** | **45°** | **0.71（71%）** | **+41%** |

→ **WBは最小化すべき。物理的下限 = タイヤ外径340mm + 40mmクリアランス = 380mm。**

### ピボットターン OK/NG 判定（現設計値）

```
各値：
  τ_stall    = 2.4 N·m（JGB37-520 ストールトルク）
  r_tire     = 0.170m
  Tread      = 0.404m
  d_wheel    = √(0.200² + 0.202²) = 0.284m
  φ          = arctan(400/404) ≈ 45°
  cos(φ)     = 0.711

  F_motor_1輪= τ_stall / r_tire = 14.1 N
  τ_motor    = 4 × 14.1 × 0.202 = 11.4 N·m

車重別判定（WB=400mm）：
  τ_friction = μ × W × 0.284

  μ=0.3, W=10.5kg（103N）: τ_friction = 8.8 N·m  < 11.4 → ✅ OK（余裕26%）
  μ=0.4, W=10.5kg（103N）: τ_friction = 11.7 N·m > 11.4 → ❌ NG（3%不足）
```

| μ（草地） | ピボターン可能最大車重 | 実重量10.5kgでの判定 |
|---|---|---|
| 0.2（湿った軟草） | 20.4 kg | ✅ 余裕大 |
| 0.3（標準草地） | **13.6 kg** | ✅ 余裕29% |
| 0.35（中間） | **11.7 kg** | ✅ ギリ |
| 0.4（乾燥硬草） | **10.2 kg** | ❌ 0.3kgオーバー |

### ⚠️ 設計の割り切り

乾燥した固い草地（μ≥0.4）でのピボットターンは困難。  
→ **行端転換は差動旋回（R=1〜2m）または3点ターンで対応。ピボットターンは不要件とする。**

草刈り環境（刈り払い後の草・朝露あり）の実態は μ=0.25〜0.35 が現実的。

## 重心（CG）・バッテリー配置

### バッテリーなし本体 CG

| 部品 | 質量 | 前輪軸からの距離 | モーメント |
|---|---|---|---|
| チップソー一式 | 750g | 0mm | 0 |
| 前輪モーター×2 | 600g | 0mm | 0 |
| 後輪モーター×2 | 600g | +400mm | 240,000 |
| タイヤ+ホイール×4 | 3,800g | +200mm | 760,000 |
| フレーム | 1,500g | +200mm | 300,000 |
| 基板・ESC・配線 | 750g | +200mm | 150,000 |
| **本体小計** | **8,000g** | **CG = 181mm** | 1,450,000 |

WB=400mm の中点は 200mm → **本体は19mm前寄り**。

### 50/50バランス達成のためのバッテリー配置

```
(8,000 × 181 + M_bat × X_bat) / (8,000 + M_bat) = 200

Makita 互換 6Ah × 2（M_bat = 1,800g）:
  X_bat = (200 × 9,800 − 8,000 × 181) / 1,800
        = (1,960,000 − 1,448,000) / 1,800 = 284mm

Makita 互換 9Ah × 2（M_bat = 2,600g）:
  X_bat = (200 × 10,600 − 8,000 × 181) / 2,600
        = (2,120,000 − 1,448,000) / 2,600 = 258mm
```

| バッテリー | 質量 | 配置位置（前輪軸から） | 後輪軸からの距離 | 走行時間概算 |
|---|---|---|---|---|
| Makita 互換 6Ah × 2 | ~1,800g | **+284mm** | 116mm手前 | ~2時間 |
| Makita 互換 9Ah × 2 | ~2,600g | **+258mm** | 142mm手前 | ~3時間 |

→ どちらも前後軸間に収まる。**Ver1はバッテリー締結穴位置を計算値で確定する（調整機構なし）。**

## ステアリング方式の比較

**アーティキュレートステアリングを本命とする。** スクラブによる草地ダメージが少なく、農地での実用性が高いと判断。スキッドステアはフォールバック。

### 3方式の比較

| 項目 | **アーティキュレート（本命）** | スキッドステア（フォールバック） | 差動二輪+キャスター |
|---|---|---|---|
| タイヤスクラブ | **少** | 多（差動二輪比+41%） | 最小 |
| 最小旋回半径 | ~400〜600mm | ゼロ（その場回転） | ゼロ（その場回転） |
| 草へのダメージ | **小** | 大 | 最小 |
| フレーム構造 | **2分割＋センターピン** | 一体 | 一体 |
| 操舵アクチュエーター | **必要（60 kg·cm 級）** | 不要 | 不要 |
| 4WDとの相性 | ○（前後フレーム独立駆動） | ◎ | △（後2輪のみ） |
| 直進安定性 | △（IMU補助要） | △（IMU補助要） | △（IMU補助要） |
| 行端旋回 | 切り返し（最小R固定） | 差動旋回（R=1〜2m）で対応 | その場回転 |
| チップソー最前端 | ✅ | ✅ | ✅ |
| Ver1適性 | **○** | ○ | ○ |

### アーティキュレートステアリングの詳細

```
【上面図：直進時】
  [前フレーム: チップソー + 前輪×2]─●─[後フレーム: バッテリー + 後輪×2]
                                    ↑
                                センターピン（WBの中点 = 前輪軸から+200mm）

【上面図：左旋回時（最大舵角±30°）】
     [前フレーム]
            ╲← 前後フレームが折れ曲がる
             ●
            ╱
     [後フレーム]
```

**センターピントルクの計算（停車中・最悪ケース）：**

```
τ_pin = μ × W × (WB/2)
      = 0.3 × 103N × 0.2m
      = 6.2 N·m ≈ 63 kg·cm
```

**緩和策：**

| 緩和策 | 効果 | 備考 |
|---|---|---|
| ① 走行しながら操舵（≥0.5 km/h） | 動摩擦に移行しトルク 30〜50% 減 | 制御上の制約として設定可能 |
| ② DCモーター + ウォームギア（60 kg·cm 級） | 停車操舵も対応 | JGB37-520 の流用も検討可 |
| ③ センターピンをリア寄りにオフセット | 前フレームの力腕を短縮 | 旋回半径が非対称になる |

**「走行しながら操舵」する場合のトルク試算：**

```
速度 0.5 km/h での動摩擦係数 ≈ 0.2（草地・転がり中）
τ_pin ≈ 0.2 × 103N × 0.2m = 4.1 N·m ≈ 42 kg·cm
→ 60 kg·cm 級モーターで対応可能
```

**採用時に確定が必要な事項：**
- ピン位置（前後重量バランスと旋回半径に直結）
- 操舵アクチュエーター選定（60 kg·cm 級DCモーター or JGB37-520 流用）
- 旋回時の前後輪速度差の制御方式（内外輪速度差の計算）
- 操舵角度範囲（±30°が一般的な上限）

## フォールバック判断基準

**アーティキュレート → スキッドステアに戻す条件：**
- センターピンアクチュエーターの調達・設計が困難と判明した場合
- フレーム2分割構造の重量・コスト増が許容範囲を超えた場合
- 走行テストで切り返し旋回が畝間作業で実用上問題になった場合

**スキッドステア → 差動二輪＋キャスターに移行する条件：**
- 前輪軸上のパッケージング（CSモーターと駆動モーターの競合）が3Dモデルで解決不能
- 全長が850mmを超えて畝間通路への進入が困難になる

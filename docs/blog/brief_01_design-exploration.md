## 1. なぜ草刈りロボットを作るのか（動機・背景）

畑の畝間を歩くたびに草刈りが面倒と感じていた。市販の自律走行草刈りロボットはあるが、農業用の畝間（幅600〜700mm）を走れるものはほとんどない。

**作りたい理由：**
- 畝間600〜700mmの狭い通路を走れる
- コンクリートではなく草地・土の上を走る
- 市販品ではなく、自分で設計してどう動くかを理解したい

**Ver1の割り切り：**
- 自律走行はIMU＋オドメトリ（GPSなし）
- DUALSHOCKで手動操作もできる
- まず走らせることを最優先

---

## 2. 最初の設計問題：どう曲がるか

草地・狭い畝間を走るには「旋回方式」の選択が核心だった。検討した3方式：

### スキッドステア（最終採用候補）
左右の車輪速度差だけで旋回する。ステアリング機構が不要でシンプル。
農機・建機でよく使われる。ただし旋回時にタイヤが横ずれ（スクラブ）する。

### アッカーマンステアリング（ボツ）
普通の車と同じ。前輪を曲げて旋回する。自然なハンドリングだが、前輪ステアリング機構が複雑になる。Ver1には過剰。

### アーティキュレートステアリング（検討中）
前後フレームをヒンジで結合し、折れ曲がって旋回する。建設機械（ホイールローダー等）で使われる。タイヤスクラブが少ない。ただしセンターピンに旋回トルクが集中する問題がある。

## 3. 旋回方式の比較（決定中）

現在スキッドステアで進めているが、走行テスト後にアーティキュレートに移行する可能性もある。

| 方式 | タイヤスクラブ | 最小旋回半径 | フレーム複雑度 | Ver1適性 |
|---|---|---|---|---|
| スキッドステア | 多（差動二輪比+41%） | ゼロ（その場） | シンプル | ◎ |
| アーティキュレート | 少 | ~400〜600mm | 中（2分割＋ピン） | △ |
| 差動二輪+キャスター | 最小 | ゼロ | やや複雑 | ○ |

## 4. 設計で発見した面白い話：分力で考える旋回トルク

**ここが記事のいちばんのネタになると思います。**

スキッドステアでその場旋回（ピボットターン）をするとき、モーターが出す力のうち「実際に回転に使われる成分」は思ったより少ない。

### 直感的な説明

タイヤは縦方向にしか転がれない。ピボットターンで右前タイヤが「回転の接線方向」に動こうとするとき、タイヤの転がり方向とは角度φ（ファイ）だけずれている。

```
               ●（車両中心）
              ╱
      d=284mm╱
            ╱ φ≈45°（WB=Tread=400mmのとき）
           ╱
──────────╱──────────── タイヤ転がり方向
```

モーター力のうち接線成分（=回転に有効）は **cos(φ)** 倍。残りはスクラブ（横ずれ）になる。

```
φ = arctan(ホイールベース ÷ トレッド)
有効成分 = F_motor × cos(φ) = F_motor × Tread/2 ÷ d_wheel
```

今の設計（WB=400mm、トレッド=404mm）では φ≈45°、モーター力の **71%しか旋回に使われない**。

| WB | φ | モーター有効率 |
|---|---|---|
| 0mm（差動二輪） | 0° | 100%（最高） |
| 400mm（現状） | 45° | 71% |

**だからホイールベースは短くするほどいい。** ただし前後タイヤが接触しない物理的下限がある（タイヤ外径340mm → 下限WB約380mm）。

地面摩擦の抵抗はこの分力ロスなしに全量が抵抗になるため、モーターと摩擦は非対称な戦いをしている。

## 5. チップソー配置の発見

チップソー（刈刃）は「タイヤが草を踏む前に刈れるように、車両の最前端」に置く必要がある。

最初は「前に突き出すしかない」と思っていたが、寸法を計算すると意外な解が出た。

**前輪タイヤの内側にチップソーを収められる：**

```
タイヤ内縁  164mm
チップソー外縁 115mm
          ↓
クリアランス 49mm ← 収まる！
```

```
←────────── 480mm ──────────→
┌──────────────────────────────┐
│  ┌───┐  ┌──○──┐  ┌───┐  │
│  │タイ│  │ø230mm│  │タイ│  │
│  │ヤ  │  │チップ│  │ヤ  │  │
│  └───┘  └──────┘  └───┘  │
│  ↑前輪軸位置（チップソー中心も同位置）│
└──────────────────────────────┘
```

チップソー（径230mm）が前輪タイヤ（内縁164mm）の内側に横方向で収まり、かつ刃の先端が前輪軸より115mm前に出るので「最前端」の要件も満たせる。

全長が715mmに収まり、畝間通路（600〜700mm幅）の対角で通過できる見込み。
